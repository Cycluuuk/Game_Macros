Папка CS GO содержит макросы:
Bhop.ahk - Этот макрос отвечает за многоразовое нажатие пробела при зажатом пробеле.  Включается и выключается на F6, он может сопровождаться звуковым оповещением для этого надо назвать файлы Sound_Off.mp3 и Sound_On.mp3.
clicker.ahk - Быстрое нажатие на лкм, включается и выключается на F6.
Point screen.ahk - При нажатие на боковую клавишу мыши по центру  экране появляется красная точка.
Папка Dead by Daylight содержит макросы:
Fast clik a&d.ahk - быстрое нажатие на a d при нажатие кнопки o.
Папка Warcraft3 содержит макросы:
Qick_kast.ahk - на быстрые касты на q, w, e, r, t, d, f, g.
Qick_kast_f1.ahk - на быстрые касты на q, w, e, r, t, d, f, g, с дополнением нажимает на f1 для того что-бы переходило на героя.
RKM clicker.ahk - быстрые нажатия на пкм для быстрого перемещения.
